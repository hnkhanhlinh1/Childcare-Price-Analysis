# Childcare-Price-Analysis

Childcare Price Analysis in the United States

Author: Linh Huynh
Course: DSC640 – Data Presentation & Visualization, Bellevue University

1. Project Overview

This project analyzes childcare prices in the United States from 2008–2018 to understand how costs vary by age group and provider type.

The goal is to present childcare cost trends using clear charts, visuals, and an infographic so parents and community groups can easily understand the findings. 


2. Dataset

National Database of Childcare Prices

The dataset includes childcare prices for:

Infants

Toddlers

Preschoolers

across different provider types and years. 


3. Methods

The analysis includes:

Data cleaning and preparation

Exploratory data analysis (EDA)

Data visualization using charts and graphs

Creation of an infographic to communicate insights

Visualizations include:

Histograms

Boxplots

Line charts

Bar charts

4. Key Findings

Most childcare prices range between $100 and $150 per week.

Infant care is the most expensive across all years.

Family childcare homes (FCC) tend to be more affordable.

Childcare prices increased steadily from 2008 to 2018.
